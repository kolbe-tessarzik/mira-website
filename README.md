# Mira Website

This is the website for [Mira](https://github.com/FatalMistake02/mira), an open-source desktop browser.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Published Site

Download Mira for Windows, MacOS and Linux [here](https://mira.fatalmistake02.com/downloads).
